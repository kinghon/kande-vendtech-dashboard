## Draft

NVIDIA just released NemoClaw — an open-source agent platform that runs on any hardware.

It's a good development. Lowering the barrier to building agents is genuinely useful. More people will build. More will deploy. More will discover that building the agent was the easy part.

Here's what we've learned after running four agents in production since late 2024:

**The build is maybe 20% of the work.**

The other 80% is everything that happens after:

- **Silent failures.** Agents that stop working without throwing an error. They don't crash — they just quietly produce wrong outputs or skip tasks. You don't know until a human checks the result three days later.

- **Handoff failures.** When one agent passes work to another, the contract between them breaks when either side changes. There's no compiler to catch it. You find out when output disappears into a gap neither agent noticed.

- **Recalibration.** The first 90 days of any live agent are calibration. What worked in testing doesn't always hold in production. You build feedback loops, or you fly blind.

NemoClaw will make it easier to get an agent running. It won't solve any of these problems — because they're not technical problems. They're operational ones.

The tools are getting better. The gap that matters now isn't "can you build it" — it's "can you run it."

That's the gap we work in.
