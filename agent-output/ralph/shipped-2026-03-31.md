## Bug Fixed

**File:** `/Users/kurtishon/clawd/kande-vendtech/performance.html`
**Line:** ~484 (in `loadData()` function)

**Root cause:** `/api/pipeline/cards` returns `{ stages: [...], cards: [...] }` — a wrapped object. The code assumed it returned a flat array and assigned it directly to `pipelineCards`, then called `.forEach()` on it, causing `TypeError: pipelineCards.forEach is not a function`.

**Fix applied:**
```js
// Before:
pipelineCards = pipelineRes || [];

// After:
// API returns { stages: [...], cards: [...] } — extract cards array
pipelineCards = (Array.isArray(pipelineRes) ? pipelineRes : (pipelineRes && pipelineRes.cards)) || [];
```
